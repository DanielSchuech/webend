Here you can install new WebEnd Plugins. The package to be installed has to be a npm module. Its the same process like installing a npm mpodule via npm install (without writing npm install). You can choose between the following sources:
  • Registry: Simply add the plugin name into the input field! (Version can be added via @)
  • GIT: Paste the repository url into the input Field! URL has to be in the following form: <protocol>://[<user>[:<password>]@]<hostname> [:<port>][:][/]<path>[#<commit-ish>]
  • Tarball: Paste a url to the tarball or upload it.
